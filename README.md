## 介绍

VSpace 是一个在浏览器侧边栏上实现的垂直式的书签和标签页管理器扩展，灵感来源于 Arc 浏览器，目的是在非 Arc 浏览器上（主要是 Chrome）做到更好的的侧边栏体验。

## 功能

- 类 Arc 的书签和标签页显示
- 树状嵌套标签页
